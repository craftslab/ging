## Introduction

*pipeflow* is a pipeline orchestrator for CI/CD written in Go.

- See *[pipecli](https://github.com/craftslab/pipecli/)* as CLI of *pipeflow*.



## Prerequisites

- Go >= 1.16.0
- gRPC >= 1.36.0



## Build

```bash
git clone https://github.com/craftslab/pipeflow.git

cd pipeflow
make build
```
